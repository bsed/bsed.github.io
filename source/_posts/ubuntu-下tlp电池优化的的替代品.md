你可以通过 Slimbook PPA 下载Slimbook Battery 3。

添加 PPA 仓库
```bash
sudo add-apt-repository ppa:slimbook/slimbook
```
下载
```bash
sudo apt update && sudo apt install slimbookbattery
```
